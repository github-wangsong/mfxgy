# nest.js

## 快速开始

全局安装官方CLI

```sh
npm i -g @nestjs/cli
nest new project-name
```

运行项目
```sh
pnpm i
pnpm start:dev
```
项目目录
src
  ├── app.controller.spec.ts
  ├── app.controller.ts
  ├── app.module.ts
  ├── app.service.ts
  ├── main.ts


创建user 模块
```sh
nest g modul user
```


## 编程思想

### fp和odo

- 函数式编程 fp
- 面向对象编程 odo

### AOP
- AOP是Aspect Oriented Programming的缩写，即面向切面编程。

### IOC和DI
- IOC是Inversion of Control的缩写，即控制反转。
- DI是Dependency Injection的缩写，即依赖注入。
